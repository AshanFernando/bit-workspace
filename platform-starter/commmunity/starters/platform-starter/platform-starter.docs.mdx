## Using the platform-starter Workspace Starter

How to use this generator locally, essentially for development purposes:

```js
bit new platform-starter my-platform-starter-workspace --load-from /Users/me/path/to/this/dir --aspect teambit.commmunity/starters/platform-starter
```

How to use this generator after exporting to a remote scope:

```js
bit new platform-starter my-platform-starter-workspace --aspect teambit.commmunity/starters/platform-starter
```
